A function that uses states to do different things

Feb 21, 2022
4 minute read

This week I have been learning about States including the use in computers. I have been tasked three tasks to do in the week (automatic door, vending machine, robot), these were simple to imagine but slightly harder to implement. My favorite task was the vending machine task because it was simple but hard with the nature of it. Compared to the door/robot one which doesn’t work with python as easily as it is hard to have changing inputs and waiting time to simulate with.

The task automatic door was requiring me to make a python program that would accept input (like a remote button) and do different actions on it. An example of this is opening it, it first waits for the button to be pressed, then it checks the state, then it opens in 5 seconds, and making sure censor is not pressed.
