The APIs that are notable are the ones that can modify pages (only because it can be funny). But the website that was made [link code] was made with like no effort, this is shown by how it looks ugly, and how the code it just what the pages we copied from had it. Below shows what happens when there are lots of the same ID’s and therefore, I had to increment the name, so it didn’t clash with other parts. However, the below code one of the things that is cool, because it can be used to copy and paste simply and easily. Just need to keep WCAG in mind, because otherwise screen readers or devices without JS could have a broken page, and we don’t want that.

<div id="div1">
    <p id="p1">This is a paragraph.</p>
    <p id="p2">This is another paragraph.</p>
</div>

<script>
    const para2 = document.createElement("p");
    const node2 = document.createTextNode("This is new.");
    para2.appendChild(node2);

    const element3 = document.getElementById("div1");
    element3.appendChild(para2);
</script>
